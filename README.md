# ★ REED Design Inc. — Engineering (Prague)

Tento GitHub profil je záměrně bez veřejných repozitářů.
Většina vývoje probíhá v neveřejných kódbázích (NDA / interní projekty) a v produkčních prostředích, kde je klíčová stabilita, auditovatelnost a dlouhodobá udržitelnost.

## Primární zaměření
- **React + TypeScript**: produkční frontend, komponentová architektura, DX, výkon, integrace API
- **Mobilní aplikace**: **iOS (Swift/Xcode)** a **Android (Kotlin/Android Studio)**, včetně widgetů a notifikací
- Backend / API: Kotlin/Ktor, PHP — návrh rozhraní, validace, verzování, provozní disciplína
- Data & výkon: optimalizace dotazů, latence/propustnost, práce s velkými datasety
- Automatizace & tooling: Rust/Python, spolehlivé pipeline, integrace a workflow
- Provoz: Docker, CI/CD (GitHub Actions), Azure

## Veřejný kód
Veřejné ukázky zde nepublikuji. Pro ověřování práce preferuji cílený postup bez porušení důvěrnosti:
- anonymizované výřezy (read-only) / samostatné moduly
- technický walkthrough architektury a trade-offů nad konkrétním problémem
- code review na izolované části kódu
